Ide kerülnek a jármű képek, a jármű MODELL nevéből képezve, kisbetűvel:

  army1.png   (a Config.Factions-ban megadott 'model' mező alapján)

Ha egy adott járműhöz nincs kép, a UI automatikusan egy FontAwesome autó
ikonra vált vissza - tehát nem törik el semmi, ha még nincs kész grafikád.

Ezt a mappát az nb_shops (vehicle shop) is használja a `nui://nb_ownvehicles/
html/assets/imgs/{modell}.png` útvonalon keresztül - tehát elég csak IDE
feltölteni a képeket, mindkét helyen megjelennek.
